  [Summary]
    Unity Terrain modifier.
--------------------------------------------------------------------------
  [Demand]
    Modify(up or down) the Unity terrain heightmap.
--------------------------------------------------------------------------
  [Environment]
    Unity 5.0 or above.
    .Net Framework 3.0 or above.
--------------------------------------------------------------------------
  [Achieve]
    TerrainModifier : Draw extend editor window and modify(up or down)
    the terrain heightmap data.
--------------------------------------------------------------------------
  [Usage]
    Find the menu item "Tool/Terrain Modifier" in Unity editor menu bar
    and click it or press key combination Alt+T to open the editor window.

    Set the target "Terrain Data" and "Raise Height".

    Click the "Modify" button to open the dialog and make your choice.
--------------------------------------------------------------------------
  [Warning]
    "Modify" operate can not be recovered, make sure you have a backup
    of target terrain data.

    Don't set a negative value to the Raise Height unless you know the
    terrain data inside out.
