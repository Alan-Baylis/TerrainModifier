  [Summeray]
    This tool can be used to modify(up or down) Unity terrain heightmap.
--------------------------------------------------------------------------
  [Environment]
    This tool applies to Unity 5.0, .Net Framework 3.0 or above version.
--------------------------------------------------------------------------
  [Usage]
    Find the menu item "Tool/Terrain Modifier" in Unity editor menu bar
    and click it or press combination key Alt+T to open the editor window.
    Set the target "Terrain Data" and "Add Height".
    Click the "Modify" button to open the dialog and make your choice.
--------------------------------------------------------------------------
  [Warning]
    "Modify" operate can not be recovered, make sure you have a backup of
    target terrain data.
    Don't set a negative value to the "Add Height" unless you know the
    terrain data inside out.
